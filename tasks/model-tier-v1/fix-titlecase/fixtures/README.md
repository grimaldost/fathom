# textkit

Title-case a string.

- `title_case(s)` — upper-case the first letter of each whitespace-separated word and
  lower-case the rest (`"hELLO wORLD"` → `"Hello World"`); the empty string maps to
  `""`. See the `title_case` docstring for the full contract.

Run the tests: `python -m unittest discover -s tests -t .`
